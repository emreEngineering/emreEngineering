- 🍴 **Fork** edin
- 🌿 **Branch** oluşturun (`git checkout -b feature/AmazingFeature`)
- 💻 **Geliştirme** yapın
- 📝 **Commit** edin (`git commit -m 'Add some AmazingFeature'`)
- 🔄 **Push** edin (`git push origin feature/AmazingFeature`)
- 🔀 **Pull Request** oluşturun
